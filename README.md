# WHMCS MX Changer

A WHMCS module for managing and changing MX records.

## Description

This module allows WHMCS administrators to manage MX record configurations for domains.

## Installation

1. Upload the module files to your WHMCS installation
2. Navigate to Setup > Addon Modules in WHMCS admin
3. Activate the module
4. Configure the module settings

## Requirements

- WHMCS 8.0 or higher
- PHP 7.4 or higher
